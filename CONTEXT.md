# 关键决定

- 继续保留 Vite + 原生 JavaScript，先保证功能直接可用。
- 本地文件采用拖拽和文件选择两种入口，减少用户操作成本。
- 本地文件模式不写入 URL，只显示当前文件说明；远程模式继续保留分享链接能力。
- 首屏改成全屏引导页，避免用户一进来就看到复杂控制项。
- 侧边栏改成场景加载后再显示，并提供折叠按钮，减少遮挡画面。
